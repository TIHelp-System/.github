## 💡 Integração com Inteligência Artificial

O grande diferencial do **TIHelp** é a utilização da API da **OpenAI** para auxiliar o trabalho técnico. O sistema analisa o título e a descrição do chamado e sugere automaticamente soluções baseadas em IA.

* **Agilidade:** O técnico recebe sugestões prontas para resolver o problema.
* **Praticidade:** Com a sugestão em mãos, o técnico pode validar e enviar diretamente para o colaborador, economizando tempo de pesquisa.

---

## 🏗️ Plataformas e Perfis de Acesso

O sistema foi estruturado para que cada tipo de usuário tenha a ferramenta ideal:

### 🌐 Web (C# .NET MVC)
* **Administrador:** Gestão de usuários, relatórios e controle do sistema.
* **Técnico:** Atendimento de chamados com o auxílio da **IA**.
* **Colaborador:** Abertura e acompanhamento de chamados.

### 🖥️ Desktop (C# .NET)
* **Técnico e Colaborador:** Aplicação dedicada para Windows, facilitando o suporte interno e o diagnóstico rápido.

### 📱 Mobile (React + Expo Go)
* **Colaborador:** Versão mobile para abertura de chamados e consulta de status com total mobilidade.
